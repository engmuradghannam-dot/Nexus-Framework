## متبقٍ من تقرير المراجعة (لم يُوزَّع بعد)

- تدوير/إلغاء توكن GitHub المكشوف سابقاً في المحادثات — إجراء يدوي على GitHub، خارج نطاق الكود.
- تنظيف تاريخ Git من كلمة المرور القديمة `ghannam2020` (تصحيح الكود الحالي تم، لكنها لا تزال بتاريخ commits سابقة). عملية حساسة (rewrite history + force-push) — تحتاج موافقة صريحة قبل التنفيذ.
- تفعيل `CompanyScopedMixin` الكامل على `InvoiceViewSet` (حالياً محمي بـ tenant فقط، بدون عزل على مستوى الشركة) — تغيير سلوك صلاحيات، يحتاج مراجعة منفصلة قبل التفعيل حتى لا يقفل وصول مستخدمين حاليين.
- مراجعة متخصصة لصحة منطق المحاسبة/التصنيع تحت الحمل الفعلي (تدقيق خارجي، ليس اختبارات آلية).
- صفحات أخرى لسا موصولة بمخزن السجلات العام بدل نماذجها الحقيقية (نفس المشكلة اللي حللناها بالمبيعات/المشتريات): CRM، HR، Assets، Inventory، Warehouses — لكل وحدة منطق backend حقيقي وموثّق بالاختبارات (عدا Assets/Inventory/Warehouses) بينتظر واجهة حقيقية توصله بالمستخدم.
- وحدات أصغر بلا تغطية اختبارات بعد: assets، depreciation، banking، automation، notifications، hr_extras، attendance، pricing، uom، customfields، records، rbac، reports، trade، currencies، i18n، twofa، tenants.
