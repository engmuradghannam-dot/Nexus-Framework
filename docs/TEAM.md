## متبقٍ من تقرير المراجعة (لم يُوزَّع بعد)

- تدوير/إلغاء توكن GitHub المكشوف سابقاً في المحادثات — إجراء يدوي على GitHub، خارج نطاق الكود.
- تنظيف تاريخ Git من كلمة المرور القديمة `ghannam2020` (تصحيح الكود الحالي تم، لكنها لا تزال بتاريخ commits سابقة).
- اختبارات لوحدات: المخزون، التصنيع، المبيعات، المشتريات، الموارد البشرية.
- code-splitting للواجهة (الحزمة حالياً ~756 كيلوبايت بعد الضغط).
- مراجعة متخصصة لصحة منطق المحاسبة/التصنيع تحت الحمل الفعلي.
